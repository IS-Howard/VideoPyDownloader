# 簡介

影片下載工具 For [動畫瘋](https://ani.gamer.com.tw/), [Anime1](https://anime1.me/), [Gimy](https://gimy.ai/)




## 安裝

先安裝 chrome, ffmpeg

執行**'setup.bat'** (安裝python及環境)

  

## 使用方法

1. 執行**'run.bat'**


輸入sn碼可下載單集

![](./Tmp/1.gif)

輸入影片完整網址可選擇下載系列

![](./Tmp/2.gif)
